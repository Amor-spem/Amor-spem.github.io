Git简单教程

Git简单教程

使用git时的常用命令，使用方法。记录一下避免以后遗忘。
主要介绍常用的点，过于高阶的使用用的时候再说吧…

安装Git

  • Windows 直接在这里下载安装即可（也可以安装TortoiseGit UI界面，对于命令行比较陌生或者希望用中文的同学比较友好）
  • Linux 直接用 $sudo apt install git 安装就完事了

Git基本设置

  • 仓库：通过git inti可以创建一个空的仓库，也可以通过git clone克隆一个远程仓库
  • 配置：git config --list 查看当前 git 配置信息。git 的配置信息有两种，一种是全局配置，一种是本地配置针对当前仓库
  • git config -e 配置当前仓库设置
  • git config -e --global 配置当前系统的所有仓库配置
  • git config --global user.name "name" 设置用户名
  • git config --global user.email test@runoob.com 设置邮箱

    远程仓库还需要设置一些东西

  • 使用ssh-keygen -t rsa -C "xxxxx@xxxxx.com"生成sshkey，一路回车就好，文件位置在用户 > .ssh 文件夹下。其中.pub为公钥，将公钥复制到线上，例如github，完成后就有权限推送到远程库了。
  • 想要推送到远程库，还需要设置远程库的地址。使用git remote add [shortname] [url]设置远程仓库地址和别名。
    git remote -v可以查看远程远程仓库地址。

    Git常用操作

    设置好基本设置后就可以提交文件
  • git add .将所当前目录下所有文件添加到暂存区
  • git commit -m '说明'提交暂存区文件到本地仓库
  • git push <远程主机名> <本地分支名>:<远程分支名>将文件推送到远程仓库
  • git pull <远程主机名> <远程分支名>:<本地分支名>将远程文件拉取到本地仓库
  • git branch列出分支列表
  • git branch (branchname)新建分支
  • git checkout (branchname)切换不同的分支
  • git branch -d (branchname)删除分支
  • git merge (branchname)合并分支到当前分支
  • git log查看历史提交记录
  • git status查看仓库当前的状态，显示有变更的文件
  • git tag -a v1.0 提交标签
  • git reset --hard 标记值重置当前工作区到指定位置

以上基本是使用git常用命令了。

git确实非常好用，尤其是编程…

对于大文件存储有些尴尬，长时间使用会导致.git文件巨大无比…
